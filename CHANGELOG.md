# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-22

### Added
- Initial release of openclaw-apple-mail plugin
- Apple Mail integration via AppleScript
- Per-thread session isolation using stable thread IDs
- Email monitoring and polling system
- Sender allowlist for security
- HTML table extraction and processing
- Session watcher and debug utilities
- Configurable polling intervals
- Archive on reply functionality
- Thread context management
- Email channel integration for OpenClaw
- Support for multiple email accounts
- Reply policies (open, allowlist, sender-only)

### Security
- Sender allowlist enforcement
- Self-reply prevention to avoid loops
- Outbound restrictions control

### Documentation
- Comprehensive README with usage examples
- Configuration guide
- Architecture overview
- Security best practices
